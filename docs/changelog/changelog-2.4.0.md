# Changelog - v2.4.0

**发布日期**: 2026-03-27

IT Toolbox v2.4.0 从 v2.3.0 的 126 个工具扩展到 136 款工具，新增 10 个实用工具，Phase 4 开发正式启动。

---

## 新增工具（10个）

### AI 结构化提取
- 非结构化文本智能提取为结构化 JSON
- 支持自定义提取字段
- 内置常见场景模板（简历信息、产品描述、新闻文章、订单信息）
- 自动识别文本中的关键信息
- 提供提取结果说明

### AI 翻译
- 多语言翻译支持（中文、英语、日语、韩语、法语、德语、西班牙语等）
- 技术文档翻译优化
- 自动检测源语言
- 保留技术术语和代码片段
- 提供翻译备注说明

### AI 报错解释
- 粘贴错误信息，AI 解释原因和修复方案
- 支持添加上下文信息
- 提供详细的错误原因分析
- 给出多个可行的修复方案
- 支持常见编程语言和框架的错误解析

### AI 命名助手
- 描述功能自动推荐变量/函数/类名
- 支持多种命名风格：camelCase、PascalCase、snake_case、kebab-case、CONSTANT
- 支持多种类型：变量、函数、类、常量
- 提供 3-5 个命名建议
- 每个建议附带解释说明

### AI 生成 Mock 数据
- 自然语言描述数据结构
- AI 生成逼真的测试数据
- 支持指定生成数量（1-20条）
- 内置常见数据模板（用户信息、商品数据、订单信息、博客文章、公司信息）
- 一键复制 JSON 数据

### AI Shell 命令生成
- 自然语言描述生成 Shell 命令
- 同时提供 Bash 和 PowerShell 两种版本
- 危险命令警告提示
- 提供命令详细说明
- 内置常见操作示例

### 环境变量 Diff
- 两段 .env 文件对比
- 高亮显示差异（新增、删除、变更）
- 安全脱敏显示敏感值
- 支持显示/隐藏完整值
- 一键复制差异结果

### Changelog 生成
- Git log 粘贴自动格式化
- 支持 Conventional Commits 规范
- 自动按类型分组（新功能、Bug修复、文档、重构等）
- 支持自定义版本号和发布日期
- 一键复制 Markdown 格式

### JSON-RPC 测试
- 构造 JSON-RPC 2.0 请求
- 支持自定义端点 URL
- 内置常见 RPC 方法示例
- 请求历史记录
- 一键复制响应结果

### 快捷键系统
- 全局快捷键管理
- 快捷键帮助面板（Ctrl+?）
- 支持自定义快捷键注册
- 输入框智能忽略

## 技术亮点

### AI 工具（6个）
以下工具需要 Workers AI 支持：
- ai-text-extract（AI结构化提取）
- ai-translate（AI翻译）
- ai-error-explain（AI报错解释）
- ai-naming（AI命名助手）
- ai-mock-data（AI生成Mock数据）
- ai-shell-cmd（AI Shell命令生成）

### 纯前端工具（3个）
以下工具采用纯前端实现，无需后端 API 支持：
- env-diff（环境变量Diff）
- changelog-gen（Changelog生成）
- keyboard-shortcuts（快捷键系统）

### API 依赖工具（1个）
以下工具需要后端 API 支持：
- json-rpc-test（JSON-RPC测试）

### 功能特色
- AI 工具统一使用 Workers AI binding，通过 Pages Functions 调用
- ai-shell-cmd 提供危险命令警告，保障操作安全
- env-diff 支持敏感值脱敏，保护隐私
- changelog-gen 完整支持 Conventional Commits 规范
- 所有工具通过 TypeScript 严格模式检查
